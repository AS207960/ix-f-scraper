============
Installation
============


Stable release
--------------

To install IX-F Scraper, run this command in your terminal:

.. code-block:: console

    $ pip install ix_f_scraper

This is the preferred method to install IX-F Scraper, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for IX-F Scraper can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/as207960/ix_f_scraper

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/as207960/ix_f_scraper/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/as207960/ix_f_scraper
.. _tarball: https://github.com/as207960/ix_f_scraper/tarball/master
